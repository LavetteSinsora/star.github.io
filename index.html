STAR: Student–Teacher Augmented Response

Author: Xingyang (Chris) He

Mentorship: Carnegie Mellon University, Human-Computer Interaction Institute

Overview

STAR is a multi-agent LLM-based tutoring system designed to support self-directed learning. The system introduces a Student–Teacher Enactment framework to simulate user response, anticipate learner confusion, reduce repetitive prompting, and improve learning efficiency.
